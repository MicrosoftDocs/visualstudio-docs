# IDebugEngine3::SetEngineGuid
This method sets the debug engine's (DE) `GUID`.

## Syntax

```cpp
HRESULT SetEngineGuid(
   GUID* guidEngine
);
```

```csharp
int SetEngineGuid(
   ref Guid guidEngine
);
```

## Parameters
`guidEngine`\
[in] `GUID` of the engine.

## Return Value
 If successful, returns `S_OK`; otherwise, returns error code.
