# IDebugDocumentTextEvents2::onDestroy
Indicates that the entire document has been destroyed.  
  
## Syntax  
  
```cpp  
HRESULT onDestroy(   
   void   
);  
```  
  
```csharp  
int onDestroy();  
```  
  
## Return Value  
 If successful, returns `S_OK`; otherwise, returns an error code.
