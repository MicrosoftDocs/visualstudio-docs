# IDiaStackFrame::get_base
Retrieves the base address of the frame.

## Syntax

```C++
HRESULT get_base (Â 
   ULONGLONG* pRetVal
);
```

#### Parameters
 `pRetVal`

[out] Returns the base address.

## Return Value
 If successful, returns `S_OK`. Returns `S_FALSE` if the property is not supported. Otherwise, returns an error code.
