# IDiaSymbol::get_uavSlot
Retrieves the uav slot.

## Syntax

```C++
HRESULT get_uavSlot(
   DWORD* pRetVal);
```

#### Parameters
 `pRetVal`

[out] A pointer to a `DWORD` that holds the uav slot.

## Return Value
 If successful, returns `S_OK`; otherwise, returns `S_FALSE` or an error code.
