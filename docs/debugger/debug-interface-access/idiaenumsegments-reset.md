# IDiaEnumSegments::Reset

 [!INCLUDE [Visual Studio](~/includes/applies-to-version/vs-windows-only.md)]
Resets an enumeration sequence to the beginning.

## Syntax

```C++
HRESULT Reset ();
```

## Return Value
 Returns `S_OK`.
