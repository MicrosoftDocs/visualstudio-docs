# IDiaTable::get_Count
Retrieves the number of items in the table.  
  
## Syntax  
  
```cpp#  
HRESULT get_Count (   
   LONG* pRetVal  
);  
```  
  
#### Parameters  
 `pRetVal`  
 [out] Returns the number of items in the table.  
  
## Return Value  
 If successful, returns `S_OK`; otherwise, returns an error code.
