# IDiaSymbol::get_samplerSlot
Retrieves the sampler slot.

## Syntax

```C++
HRESULT get_samplerSlot(
   DWORD* pRetVal);
```

#### Parameters
 `pRetVal`

[out] A pointer to a `DWORD` that holds the sampler slot.

## Return Value
 If successful, returns `S_OK`; otherwise, returns `S_FALSE` or an error code.
