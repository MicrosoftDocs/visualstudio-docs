# IDiaSymbol::get_stride
Retrieves the stride of the matrix or strided array.

## Syntax

```C++
HRESULT get_stride(
   DWORD* pRetVal);
```

#### Parameters
 `pRetVal`

[out] A pointer to a `DWORD` that holds the stride.

## Return Value
 If successful, returns `S_OK`; otherwise, returns `S_FALSE` or an error code.
