# IDiaSymbol::get_textureSlot
Retrieves the texture slot.

## Syntax

```C++
HRESULT get_textureSlot(
   DWORD* pRetVal);
```

#### Parameters
 `pRetVal`

[out] A pointer to a `DWORD` that holds the texture slot.

## Return Value
 If successful, returns `S_OK`; otherwise, returns `S_FALSE` or an error code.
