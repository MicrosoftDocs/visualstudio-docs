# UdtKind
Describes the variety of user-defined type (UDT).

## Syntax

```C++
enum UdtKind {
    UdtStruct,
    UdtClass,
    UdtUnion,
    UdtInterface
};
```

## Elements
UdtStruct
UDT is a structure.

UdtClass
UDT is a class.

UdtUnion
UDT is a union.

UdtInterface
UDT is an interface.

## Remarks
The values in this enumeration are returned by the [IDiaSymbol::get_udtKind](../../debugger/debug-interface-access/idiasymbol-get-udtkind.md) method.

## Requirements
Header: cvconst.h
