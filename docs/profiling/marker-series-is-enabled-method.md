# marker_series::is_enabled method
Determines if any session has enabled the provider.

## Syntax

```cpp
bool is_enabled();
bool is_enabled(
   marker_importance _Importance,
   int _Category
);
```

#### Parameters
 `_Importance`
 Importance level.

 `_Category`
 Category.
