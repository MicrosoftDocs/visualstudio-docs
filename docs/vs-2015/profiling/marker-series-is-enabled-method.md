# marker_series::is_enabled Method
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Determines if any session has enabled the provider.  
  
## Syntax  
  
```  
bool is_enabled();  
bool is_enabled(  
   marker_importance _Importance,  
   int _Category  
);  
```  
  
#### Parameters  
 `_Importance`  
 Importance level.  
  
 `_Category`  
 Category.
