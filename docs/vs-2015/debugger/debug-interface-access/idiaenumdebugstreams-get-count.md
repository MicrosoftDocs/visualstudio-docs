# IDiaEnumDebugStreams::get_Count
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Retrieves the number of debug streams.  
  
## Syntax  
  
```cpp#  
HRESULT get_Count(Â
```  

#### Parameters  
 `pRetVal`
