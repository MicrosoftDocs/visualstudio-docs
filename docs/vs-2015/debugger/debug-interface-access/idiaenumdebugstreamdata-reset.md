# IDiaEnumDebugStreamData::Reset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Resets to the beginning of an enumerated sequence.  
  
## Syntax  
  
```cpp#  
HRESULT Reset ();  
```  
  
## Return Value  
 Returns `S_OK`.
