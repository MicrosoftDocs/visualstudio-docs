# -UseEnv (devenv.exe)
Starts Visual Studio and loads environmental variables into the **VC++ Directories** dialog box.  
  
## Syntax  
  
```  
Devenv /useenv  
```  
  
## Example  
 The following example starts Visual Studio and loads environment variables into the **VC++ Directories** dialog box.  
  
```  
Devenv.exe /useenv  
```
