# IDiaLineNumber::get_compiland
Retrieves a reference to the symbol for the compiland that contributed the bytes of image text.  
  
## Syntax  
  
```cpp#  
HRESULT get_compiland (   
   IDiaSymbol** pRetVal  
);  
```  
  
#### Parameters  
 pRetVal  
 [out] Returns an [IDiaSymbol](../VS_debugger/IDiaSymbol.md) object for the compiland that contributed the bytes of image text.  
  
## Return Value  
 If successful, returns `S_OK`. Returns `S_FALSE` if this property is not supported. Otherwise, returns an error code.
