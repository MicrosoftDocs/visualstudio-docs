# IJsDebugFrame::GetDebugProperty Method
Returns a property browser for this stack frame.  
  
## Syntax  
  
```cpp
HRESULT GetDebugProperty(  
   IJsDebugProperty **ppDebugProperty  
);  
```  
  
#### Parameters  
 `ppDebugProperty`  
 [out] Object representing the property browser.
