# IJsDebugBreakPoint::Enable Method
Enables the breakpoint.  
  
## Syntax  
  
```cpp
HRESULT Enable(void);  
```  
  
## Return Value  
  
## Remarks  
 Returns E_UNEXPECTED if called on a deleted breakpoint.  
  
## Requirements  
 **Header:** jscript9diag.h
