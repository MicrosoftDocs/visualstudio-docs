# IJsDebugProcess::PerformAsyncBreak Method
Puts the script engine in break mode causing it to break on the next script instruction.  
  
## Syntax  
  
```  
HRESULT PerformAsyncBreak(  
   DWORD threadId  
);  
```  
  
#### Parameters  
 `threadId`  
 [in] The thread ID.  
  
## Return Value  
  
## Requirements  
 **Header:** jscript9diag.h
