# IJsDebugFrame::GetReturnAddress Method
Gets the return address pushed at the 'start' (see GetStackRange) of the frame.  
  
## Syntax  
  
```  
HRESULT GetReturnAddress(  
   UINT64 *pReturnAddress  
);  
```  
  
#### Parameters  
 `pReturnAddress`  
 [out] The return address.
