# IJsDebugBreakPoint::Delete Method
Deletes the breakpoint.  
  
## Syntax  
  
```cpp
HRESULT Delete(void);  
```  
  
## Return Value  
  
## Remarks  
 Returns S_FALSE if called on a deleted breakpoint.

 **Header:** jscript9diag.h
