# IRemoteDebugApplicationEx:RevokeBreak

Revokes a break command.

## Syntax

```cpp
HRESULT RevokeBreak( );
```

### Parameters

None.

## Return Value

The method returns an `HRESULT`. Possible values include, but are not limited to, those in the following table.

|Value|Description|
|-----------|-----------------|
|`S_OK`|The method succeeded.|
