# IDebugPropertyEnumType_All::GetName
Returns a BSTR containing the name of the `EnumType`.  
  
## Syntax  
  
```cpp
HRESULT GetName(  
   BSTR*  pname  
);  
```  
  
#### Parameters  
 `pname`  
 [out] A BSTR containing the name of the `EnumType`.  
  
## Return Value  
 Returns a valid `HRESULT`, typically `S_OK`.
