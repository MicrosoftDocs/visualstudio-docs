# IJsDebugFrame::GetName Method
Gets the user-friendly name of the stack frame.  
  
## Syntax  
  
```  
HRESULT GetName(  
   BSTR *pName  
);  
```  
  
#### Parameters  
 `pName`  
 [out] The name of the stack frame.
