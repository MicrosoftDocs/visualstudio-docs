# SCRIPT_E_PROPAGATE Error Code
A script error is being propagated to the caller, which might be in a different thread. The host should pass the error code to the caller.  
  
## Syntax  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
