# IJsDebugStackWalker::GetNext Method
Gets the next frame.  
  
## Syntax  
  
```  
HRESULT GetNext(  
   IJsDebugFrame **ppFrame  
);  
```  
  
#### Parameters  
 `ppFrame`  
 [out] Object representing the stack frame.  
  
## Return Value  
  
## Remarks  
 Returns E_JsDEBUG_OUTSIDE_OF_VM when there are no more stack frames to be enumerated  
  
## Requirements  
 **Header:** jscript9diag.h
