# IJsDebugProperty::GetPropertyInfo Method
Gets Information for this object.  
  
## Syntax  
  
```  
HRESULT GetPropertyInfo(  
   UINT nRadix,  
   JsDebugPropertyInfo *pPropertyInfo  
);  
```  
  
#### Parameters  
 `nRadix`  
 [in] Radix to use.  
  
 `pPropertyInfo`  
 [out] Information about the object.  
  
## Return Value  
  
## Requirements  
 **Header:** jscript9diag.h
