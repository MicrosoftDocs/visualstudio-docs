# JsPropertyIdRef Typedef
A property identifier.  
  
## Syntax  
  
```  
typedef JsRef JsPropertyIdRef;  
```  
  
## Remarks  
 Property identifiers are used to refer to properties of JavaScript objects instead of using strings.  
  
## Requirements  
 **Header:** jsrt.h
