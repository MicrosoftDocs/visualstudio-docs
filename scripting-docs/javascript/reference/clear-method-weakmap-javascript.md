# clear Method (WeakMap) (JavaScript)
Removes all elements from a `WeakMap` object.  
  
## Syntax  
  
```javascript  
weakmapObj.clear()  
```  
  
#### Parameters  
 `weakmapObj`  
 Required. The `WeakMap` object to clear.
