# Compiler Error CS2035
Command-line syntax error:  Missing ':<number\>' for 'compiler_option' option  
  
 Some compiler options require a value.  
  
## Example  
 The following sample generates CS2035.  
  
```  
// CS2035.cs  
// compile with: /baseaddress  
// CS2035 expected  
```
