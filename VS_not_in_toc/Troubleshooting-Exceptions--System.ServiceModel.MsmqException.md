# Troubleshooting Exceptions: System.ServiceModel.MsmqException
An <xref:System.ServiceModel.MsmqException?qualifyHint=False> exception encapsulates errors returned by Message Queuing. This exception is thrown by the Message Queuing transport and the Message Queuing integration channel.
